# Python OpenShift Demo

This repository contains a simple Python web application designed to run on
Red Hat OpenShift using a Dockerfile-based build.

## Overview

- Language: Python 3
- Framework: Flask
- Container: Docker
- Platform: Red Hat OpenShift

The application exposes a simple HTTP endpoint and is intended for
demonstration and testing purposes.
